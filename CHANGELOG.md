# Changelog

## 1.0.0 (2026-03-13)

Initial public release.

### Features
- 14 MCP tools across institution-level and branch-level queries
- 587 BSP-supervised financial institutions with PSGC geographic enrichment (99.8% match rate)
- 37,834 geocoded branch locations (Financial Service Access Points) with 100% PSGC match rate
- 2024 Census population cross-referencing for coverage and density analysis
- Financial inclusion tools: unbanked area detection, underserved area ranking, coverage comparison
- Institution footprint mapping with fuzzy name matching
- Cloudflare Workers deployment with Durable Objects
- GitHub Actions CI/CD
